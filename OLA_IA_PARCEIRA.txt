---
PROJETO: MangaReader
---

Uma plataforma de leitura de mangás com camadas Free e Premium.
Stack:
  - Backend : Node.js + Express + tRPC + Drizzle ORM + PostgreSQL (Supabase)
  - Auth    : Clerk
  - Cache   : Upstash Redis (rate limiting)
  - Email   : Resend
  - Analytics: PostHog
  - Erros   : Sentry
  - Frontend: HTML + CSS + JavaScript puro (sem framework)

---
PADRÕES DE CÓDIGO ADOTADOS
---

- CSS: variáveis via --tokens definidos em :root no styles.css
  Ex: --purple, --bg-card, --text-1, --text-2, --border, --shadow-card
- Sem bundler no frontend: imports via <script src="..."> em ordem
- tRPC via HTTP (GET com ?input= para queries, POST para mutations)
- Auth token: window.ClerkAuth.getToken() injetado nos headers
